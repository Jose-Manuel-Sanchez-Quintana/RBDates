# RB Dates Landing Page

A premium landing page for RB Dates, featuring a responsive design, product carousel, and "Palm to Table" aesthetic.

## Tech Stack

- **React** (Vite)
- **TailwindCSS** (Styling)
- **Framer Motion** (Animations & Carousel)
- **Lucide React** (Icons)

## Setup & Run

1. Install dependencies (if not already done):
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`.

## Features

- **Hero Section**: Immersive background with parallax-like fade-in.
- **Lazy Loading**: Sections are lazy-loaded for optimal performance.
- **Custom Carousel**: Swipe-enabled product slider in the "Our Collection" section.
- **Responsive Navigation**: Mobile-friendly menu with smooth transitions.
- **Contact Form**: Clean, accessible layout.

## Customization

- **Colors**: Defined in `tailwind.config.js` (Rich Brown, Deep Date Brown, Warm Cream, Desert Beige).
- **Images**: Located in `public/assets`.
